## OSX setup

    curl -O https://raw.githubusercontent.com/haf/osx/master/Rakefile
    rake

### What it does

 - Installs xcode
 - Setup default osx settings
 - Installs brews
 - Installs casks
 - Installs zshell
 - Asks and sets computer name
 - Asks and sets git name and email
 - Installs nvm and npm
