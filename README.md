## OSX setup

`curl -O https://raw.githubusercontent.com/haf/osx/master/Rakefile && sh 'rake download' && sh 'rake prepare' && sh 'rake install'`

### What it does

 - Installs xcode
 - Setup default osx settings
 - Installs brews
 - Installs casks
 - Installs zshell
 - Asks and sets computer name
 - Asks and sets git name and email
